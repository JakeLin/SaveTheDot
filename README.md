## How to build

* Clone the repository

```bash
$ git clone https://github.com/JakeLin/SaveTheDot.git
```

* Open the project in Xcode 8

```bash
$ cd SaveTheDot
$ open "SaveTheDot.xcodeproj"
```

# Requirements

* Xcode 8
* iOS 10
* Swift 3
